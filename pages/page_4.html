Fundology


Applying modern UI design principles to an enterprise web application.

I utilized a range of wireframing, design, and analytical tools in aiding Fundology, a Chicago-based web startup where investment professionals collaborate to close deals faster.

  • 2013 - 2014

Project Research

Fundology's provides simple and secure transaction management to the financial services. In my first line of work, I created an engaging landing page worked on redesigning the internal application to inutitive and effective in work.

Wireframing

I utilized online tools to create wireframes that could be reviewed remotely with the team and clients. With AxureRP I was additionally able to test dynamic aspect of our design, mocking up various tasks that could be done within the application.

Mockups

I utilized Fireworks, Illustrator and Photoshop to create mockups for the application. Creating splices for div's simplified the implementation process for developers.

Product Development

I continued to work with Fundology throughout 2013, improving upon my initial application designs with analytical tools and creating a mobile-friendly design.

Landing Page Design

The landing page was updated to a simpler design to drive conversions. Additionally, it utilized dynamic animations, text and imagery to engage the user. A/B testing could easily be applied to the pages to better understand user traction based on page content.

Mobile Design

To make navigation easier on mobile interfaces, I added a bottom navigation bar that provided access to the higher-level functions that were most likely to be used on the move. An additional naviagion menu accessible by swiping to the right gave the user quick access to more advanced funtions.
